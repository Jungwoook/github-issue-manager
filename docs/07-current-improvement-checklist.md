# 현재 구현 범위 개선 체크리스트

## 1. PAT 연결 안정화

- [ ] PAT 등록 실패 시 GitHub 응답 메시지를 더 명확하게 표시
- [ ] PAT 등록 성공 후 저장소 자동 동기화 결과를 더 직관적으로 표시
- [ ] PAT 만료, 권한 부족, 잘못된 토큰 상태를 구분해 안내

## 2. 저장소 기능 점검

- [ ] 저장소 목록 조회와 저장소 새로고침 동작 재확인
- [ ] 저장소 없음, 권한 없음, 동기화 실패 메시지 분리
- [ ] 저장소 화면 한글 문구와 정보 표시 정리

## 3. 이슈 기능 점검

- [ ] 이슈 목록 조회와 이슈 새로고침 동작 재확인
- [ ] 이슈 생성 후 상세 화면 이동 재확인
- [ ] 이슈 수정 후 반영 결과 확인
- [ ] 이슈 닫기 후 목록 반영 확인
- [ ] 검색어, 상태 필터 동작 확인

## 4. 댓글 기능 점검

- [ ] 댓글 조회와 댓글 새로고침 동작 재확인
- [ ] 댓글 작성 후 목록 갱신 확인
- [ ] 댓글 새로고침 실패 시 원인 메시지 개선

## 5. 미지원 기능 정리

- [ ] 라벨 기능을 계속 안내 화면으로 둘지, 백엔드 구현으로 이어갈지 결정
- [ ] 현재 범위에 없는 기능은 프론트에서 호출하지 않는지 재점검
- [ ] 문서와 화면 문구에 현재 미지원 범위를 명확히 표시

## 6. UI 및 문구 정리

- [ ] 남아 있는 깨진 한글 문구 정리
- [ ] 버튼, 배너, 빈 상태 메시지 표현 일관성 정리
- [ ] 테스트용 임시 문구 제거

## 7. 테스트 및 검증

- [ ] 브라우저 기준 수동 테스트 시나리오 정리
- [ ] 주요 흐름별 확인 결과 기록
- [ ] 백엔드 API 실패 케이스 테스트 보강
