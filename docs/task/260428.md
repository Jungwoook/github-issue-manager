# 260428 작업 기록

## 1. 플랫폼 연결 구조 전환

요약: app 조립 + platform 원격 호출 중심 구조로 전환

- 문제: 업무 모듈이 connection token access와 platform gateway를 함께 의존
- 지시사항: 14번 전환 계획 1~4단계 진행, 단계별 커밋
- 작업: PAT 등록 검증을 platform으로 이동
- 작업: connection은 검증된 profile과 token 저장 담당
- 작업: 원격 호출 token/baseUrl 조회를 platform 내부로 이동
- 작업: repository / issue / comment의 connection 직접 의존 제거
- 결과: platform -> connection 중심 원격 호출 구조 확보
