# 구현 요약

## 현재 구현 범위

- GitHub PAT 연결, 저장소 조회, 이슈 조회/생성/수정/닫기, 댓글 조회/작성까지 구현되어 있다.
- 백엔드는 GitHub API 연동과 캐시 저장을 담당하고, 프론트는 이 흐름을 화면으로 제공한다.

## 반영 완료된 개선 사항

- PAT 처리와 저장 방식에 대한 보안 점검을 수행했다.
- PAT 입력 UI와 저장소 목록 화면의 사용성을 개선했다.
- 토큰 암호화 방식 강화와 관련 검증을 반영했다.

## 현재 제약

- 현재 구조는 GitHub 중심이며 다른 플랫폼 확장에는 한계가 있다.
- 라벨, 담당자, 우선순위, sub-issue, GitHub App 기반 구조는 아직 미구현 상태다.

## 다음 주요 과제

- GitHub 전용 구조를 공통 코어와 GitHub 어댑터 구조로 분리한다.
- 플랫폼 확장을 고려한 인터페이스와 데이터 모델 정리가 필요하다.
