# 현재 구현 요약

## 1. 현재 구현된 기능

### PAT 연결

- PAT 등록
- PAT 연결 상태 조회
- PAT 연결 해제

### 저장소

- 저장소 목록 조회
- 저장소 새로고침
- 저장소 상세 조회

### 이슈

- 이슈 목록 조회
- 이슈 새로고침
- 이슈 생성
- 이슈 상세 조회
- 이슈 수정
- 이슈 닫기
- 제목/상태 기준 필터링

### 댓글

- 댓글 목록 조회
- 댓글 새로고침
- 댓글 작성

## 2. 프론트 상태

- PAT 설정 화면 구현 완료
- 저장소 목록 화면 구현 완료
- 이슈 목록, 상세, 생성, 수정 화면 구현 완료
- 댓글 작성 및 댓글 새로고침 UI 구현 완료
- 라벨 화면은 현재 미지원 안내 화면으로 정리됨

## 3. 백엔드 상태

- PAT 기반 인증 및 사용자 연결 구현 완료
- 저장소/이슈/댓글 캐시 구조 구현 완료
- 저장소, 이슈, 댓글 새로고침 API 구현 완료
- GitHub API 호출 실패 시 공통 예외 응답 처리 추가

## 4. 현재 제약

- 개인 사용자 1명 기준 구조
- PAT 범위 밖 저장소는 접근 불가
- 라벨, 담당자, 우선순위, sub-issue 미지원
- GitHub App, 설치 정보, 웹훅 미사용
