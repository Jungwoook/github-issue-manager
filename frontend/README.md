# Frontend Workspace

React + TypeScript + Vite 기준의 초기 구조를 구성했습니다.

## 시작

```powershell
npm install
npm run dev
```

## 주요 경로

- `src/app`: 앱 초기화, 라우터, 전역 provider, 레이아웃
- `src/pages`: 라우트 단위 페이지
- `src/features`: 사용자 액션 단위 기능
- `src/entities`: 도메인 타입과 API 모듈
- `src/widgets`: 페이지 조합용 중간 레벨 UI
- `src/shared`: 공통 API, 타입, 상수, 유틸, UI
