# GitHub Issue Manager

루트 워크스페이스는 `backend`, `frontend`, `docs` 기준으로 구성합니다.

## 구조

- `backend/`: Spring Boot 기반 백엔드 프로젝트
- `frontend/`: 프론트엔드 프로젝트 작업 공간
- `docs/`: 백엔드/프론트엔드 공통 문서

## 문서 시작점

- 백엔드 개요: `backend/README.md`
- 프론트엔드 가이드: `docs/frontend-guide.md`
- 화면 상세 명세: `docs/screen-spec.md`
- 프론트 구조 문서: `docs/frontend-structure.md`

## 비고

- 기존 백엔드 프로젝트 파일은 `backend/`로 이동했습니다.
- 루트의 `frontend/`는 다음 단계 프론트엔드 구현을 위한 빈 작업 공간입니다.
